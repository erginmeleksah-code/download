# HIZLI-indir — Web Sürümü

## Kurulum

```
pip install flask yt-dlp python-docx openpyxl pdfplumber pdf2docx reportlab
```

## Çalıştırma

```
python app.py
```

Tarayıcıda aç: **http://localhost:5000**

## Notlar
- MP3 dönüşümü için FFmpeg kurulu olmalı
- Word↔PDF için LibreOffice önerilir: https://www.libreoffice.org
- İndirilen dosyalar otomatik silinir
